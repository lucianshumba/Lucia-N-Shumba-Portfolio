# Project Context — Lucia N. Shumba Portfolio

## Project Summary
Personal data science and analytics portfolio website for Lucia Nomatter Shumba. Single-page static site showcasing career history, projects, skills, and education. Built in one session from scratch and deployed to production.

## Current Stack
- **Frontend:** Plain HTML/CSS/JavaScript — no framework, no build tool
- **Fonts:** Google Fonts — Playfair Display (headings), Inter (body), JetBrains Mono (code/labels)
- **Image generation:** Python 3.13 + Pillow 12 (`gen_og.py`) for the OG preview image
- **PDF conversion:** PowerShell + Microsoft Word COM automation (`resume.docx` → `resume.pdf`)
- **Hosting:** Vercel (free tier, static)
- **Source control:** GitHub — `main` branch, auto-deploy on push

## Architecture Snapshot
```
Lucia-N-Shumba-Portfolio/
├── index.html        ← entire site — HTML + CSS + JS in one file (~980 lines)
├── resume.pdf        ← downloadable CV (converted from resume.docx via Word COM)
├── og-image.png      ← 1200×630 OG preview image (generated by gen_og.py)
├── gen_og.py         ← Python script to regenerate og-image.png (not committed)
├── resume.docx       ← source resume (not committed)
├── linkedin-post.md          ← portfolio launch post drafts (not committed)
├── linkedin-internship-post.md ← internship announcement drafts (not committed)
└── .ai/              ← this folder
```
